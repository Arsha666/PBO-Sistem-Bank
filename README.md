# Bank Account System (Java OOP)

## Deskripsi

Project ini merupakan mini sistem **Bank Account sederhana** yang dibuat menggunakan **Java** dengan konsep **Object Oriented Programming (OOP)**, khususnya **Encapsulation**.

Sistem ini memungkinkan pengguna untuk membuat data nasabah, melakukan deposit, melakukan penarikan uang, dan melihat saldo rekening.

## Konsep yang Digunakan

* Encapsulation (atribut dibuat `private`)
* Getter dan method untuk mengakses data
* Validation pada transaksi

## Struktur Class

Project ini terdiri dari 3 class:

* **Customer** → menyimpan data nasabah
* **BankAccount** → mengelola rekening dan transaksi
* **SistemBank** → menjalankan program utama
